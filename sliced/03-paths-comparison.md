# Choose Your Path

## Path A: UX-First ✨ — 20-25 hours
**Goal**: Beautiful, engaging app. **Difficulty**: Easy to Medium.

| Week | Features | Hours |
|------|----------|-------|
| 1 | Theme + Settings + Animations | 7h |
| 2 | Sound + Mobile Optimization | 5h |
| 3 | Leaderboard + Tutorial | 7h |
| Polish | Testing & fixes | 1-2h |

**Skills gained**: React patterns, CSS animations, Web Audio API, responsive design, accessibility, localStorage.

---

## Path B: AI-First 🤖 — 18-23 hours
**Goal**: Unbeatable strategic AI. **Difficulty**: Medium to Hard.

| Week | Features | Hours |
|------|----------|-------|
| 1 | Difficulty Levels + Basic Minimax | 5h |
| 2 | Performance + Advanced Evaluation | 4h |
| 3 | Opening Book + Polish | 4h |
| Polish | Testing & debugging | 5-8h |

**Skills gained**: Game theory, Minimax, alpha-beta pruning, memoization, heuristic design, performance optimization.

---

## Path C: Balanced 👑 — 38-48 hours
**Goal**: Professional app with great UX AND intelligent AI. **Difficulty**: Varied.

| Week | Focus | Hours |
|------|-------|-------|
| 1 | UX foundations (Theme, Settings, Animations) | 7h |
| 2 | AI basics + UX enhancement (Difficulty, Minimax, Sound) | 8h |
| 3 | AI enhancement + UX polish (Performance, Leaderboard, Mobile) | 7h |
| 4 | Integration + full testing | 8h |
| Deploy | Launch | 1h |

**Skills gained**: Full-stack web dev, professional code org, testing methodology, Git workflow, deployment.

---

## Decision Tree

```
What matters most?
    ├── Look & Feel? → Path A (UX) → 20-25h
    ├── Smart Play?  → Path B (AI) → 18-23h
    └── Both?        → Path C      → 38-48h
```
